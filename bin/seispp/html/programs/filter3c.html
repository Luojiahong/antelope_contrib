filter3c

Usage:

filter3c filter_specification < in > out [-v --help -binary]

Algorithm:

Applies a time invariant filter to one or more ThreeComponentEnsembles stored in the serialized input file read from stdin.  Writes the filtered data to stdout.

Filtering is implemented with the Antelope recursive filters as defined in man wffil(3).   The required argument, filter_specification, is a string that defines their suite of filters as used by wffil.   e.g. INT is integration and "BW 1.0 5 15.0 5" is a Butterworth bandpass filter with 5 poles on each end of the pass band and corner frequencies of 1.0 and 15.0 Hz. 

Options:

-binary - switch to binary input and output (default is text data)
--help - print the usage message
-v - send some messages to stderr
